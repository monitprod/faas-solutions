# User API Function

## How to use locally?

1. Duplicate ```payload.example.json``` and rename to ```payload.json```

2. Duplicate ```.example.env``` and rename to ```.env```

3. Configure ```.env``` file

4. Start Variables

    ``` sh
    export UA_ROOT_PATH= #user_api folder
    ```

5. Start go

    ``` shell
    $ go run cmd\local\main.go
    ```
